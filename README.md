# JPEQ-地震情報 
## これについて
最新地震情報と地震情報発表順5件がみられます。

主にTurbowarpのHTTPという拡張機能を用いて受信しています。インターネットが接続されていれば、(API元にもよりますが)地震情報が表示されます。

クラウド変数を使っていないので、Turbowarpのクラウドサーバーを気にせず使えます。
